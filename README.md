# EASM — Evolutional Assembly

EASM — интерпретируемый 64-битный ассемблерный язык, программа которого может вставлять, удалять и изменять собственные исполняемые инструкции. Репозиторий содержит TypeScript VM, CLI и браузерную лабораторию.

## Быстрый старт

Требуется Node.js 20 или новее.

```powershell
npm install
npm test
npm run dev
```

Vite напечатает адрес браузерной лаборатории, обычно `http://localhost:5173`.

Страница случайного поиска программ доступна по адресу `http://localhost:5173/generator.html` или по ссылке «Генератор программ» в лаборатории.

Запуск файла через CLI:

```powershell
npx tsx src/cli.ts examples/self-insert.easm --memory 64 --steps 100 --seed 7
```

## Минимальный пример

```asm
mov 40, A
mov 2, B
add A, B, C
mul C, 10, [0]
halt
```

Операнды-источники могут быть константами, регистрами `A`–`D`, прямыми ячейками `[index]` и косвенными адресами `[A]` / `[[index]]`. Назначение — регистр или любой вид ячейки памяти. Все результаты нормализуются до знакового int64.

## Браузерная лаборатория

- редактор EASM и встроенные примеры;
- открытие и сохранение `.easm` файлов, включая повторную запись в ранее открытый файл;
- запуск в отдельном Web Worker, пошаговое исполнение и сброс;
- seed и лимит инструкций;
- регистры, память и итоговый исполняемый код;
- представления входа/выхода: UTF-8, биты и целые числа;
- журнал `insert` / `delete` / `change` и трасса команд;
- пакетный прогон до 20 входов для сравнения экспериментов.
- отдельный многопоточный генератор случайных программ с фильтрацией по ожидаемой выходной памяти, настраиваемыми границами кода и памяти.

UTF-8 кодек намеренно хранит один байт в одной 64-битной ячейке. Битовый кодек упаковывает каждые 64 последовательных бита в одну ячейку.

## Структура

```text
src/core/       parser, opcode registry, codecs, VM
src/web/        React UI и Web Worker
src/cli.ts      запуск из терминала
examples/       демонстрационные программы
docs/           спецификация языка
```

Полная семантика языка, таблица индексов операций и формат дескрипторов находятся в [docs/language.md](docs/language.md).
Описание случайного поиска находится в [docs/generator.md](docs/generator.md).
